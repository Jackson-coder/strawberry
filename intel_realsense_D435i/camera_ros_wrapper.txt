1.在工作空间下下载仓库
git clone https://github.com/IntelRealSense/realsense-ros.git
git clone https://github.com/pal-robotics/ddynamic_reconfigure/branches.git
2.编译
catkin_make
3.测试安装是否成功
roslaunch realsense2_camera rs_camera.launch
rostopic list
4.rviz显示
Terminal 1:roslaunch realsense2_camera rs_camera.launch filters:=pointcloud
Terminal 2:rosrun rviz rviz
在rviz中增加pointcloud2选项
